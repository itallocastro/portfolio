Tecnologias

Angular

Framework para construção de aplicações que utiliza: HTML, CSS e Typescript.Muitos utilizados quando o assunto é single page application.

Utilizo essa tecnologia para o desenvolvimento frontend de aplicações web.

Express js

Framework para Node js com muitos recursos para o desenvolvimento web.

Utilizo essa tecnologia para a criação de api's que servem nossa aplicação web.

Node js

Ambiente de execução javascript, onde podemos executar nossa aplicação sem necessariamente precisar de um browser.Apresenta alta capacidade de escalabilidade.

Flutter

Framework utilizado pelo google para criação de aplicações mobile em multiplataformas(android/iOS), tem como linguagem principal o Dart.

Utilizo essa tecnologia para a criação de aplicações mobile.
